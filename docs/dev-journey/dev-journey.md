# 커피 쇼핑몰 하나가 완성되기까지, 10개의 정거장

코드를 한 줄도 몰라도 괜찮아요. 이 문서는 **라오스 유기농 커피 쇼핑몰(Lao Aroma)** 을 실제로 만들면서 거쳐온 과정을 순서 그대로 기록한 거예요. 각 단계마다 "실제로 무엇을 했는지"와 "일상 속 무엇과 비슷한지" 비유를 같이 적었어요.

- 총 10단계
- 상품 화면 8종
- 실제로 배포된 사이트 1개 → https://laos-coffee.vercel.app

## 1. 설계도(PRD) 확인하기

**한 일** — 어떤 쇼핑몰을 만들지 정리해둔 "PRD"라는 기획 문서를 먼저 읽었어요. 누구에게 팔지, 어떤 화면이 필요한지, 상품 정보는 뭘 보여줄지가 다 적혀 있었어요.

**비유** — 집을 짓기 전에 설계도부터 펼쳐보는 것과 같아요. 설계도 없이 벽부터 쌓으면 나중에 다 부수고 다시 지어야 하거든요.

`PRD 문서` `기획`

## 2. 빈 창고 열기 (저장소 클론)

**한 일** — GitHub이라는 곳에 미리 만들어둔 빈 저장소를 내 컴퓨터로 그대로 복사해왔어요. 아직 안에는 아무것도 없는 빈 창고 같은 상태였어요.

**비유** — 물건을 만들기 전에 물건을 보관할 창고 열쇠부터 받아오는 것과 비슷해요.

`Git` `GitHub` `clone`

## 3. 뼈대 세우기 (프로젝트 생성)

**한 일** — "Next.js"라는 도구로 웹사이트의 기본 뼈대를 만들었어요. 화면을 그리는 방식, 디자인 도구, 글자·숫자를 미리 확인하는 도구까지 기본 골조를 한 번에 세팅했어요.

**비유** — 건물을 지을 때 기둥과 골조부터 세우는 것과 같아요. 골조가 튼튼해야 그 위에 벽과 지붕을 올릴 수 있어요.

`Next.js` `TypeScript` `Tailwind CSS`

## 4. 가짜 상품 준비하기 (목업 데이터)

**한 일** — 아직 진짜 판매 시스템이 없어서, 원두·드립백·커피믹스 등 상품 8종의 정보를 미리 만들어뒀어요. 가격, 원산지, 후기 같은 정보도 함께요.

**비유** — 매장을 열기 전에 진짜 상품 대신 견본품이나 모형을 진열대에 먼저 놓아보는 것과 같아요.

`목업 데이터` `TypeScript 타입`

## 5. 진열대 만들기 (상품 목록 페이지)

**한 일** — 8개 상품을 카드 형태로 늘어놓는 화면을 만들었어요. 사진 자리, 이름, 가격, 할인율, 별점, BEST/NEW 표시, 품절 표시까지 한 화면에서 볼 수 있게 했어요.

**비유** — 매장에 들어갔을 때 가장 먼저 보이는 진열대를 꾸미는 작업과 같아요.

`화면 컴포넌트` `반응형 그리드`

## 6. 상세 설명 카드 만들기

**한 일** — 상품을 클릭하면 원산지, 무게, 유기농 인증 소개, 다른 사람들의 후기, "장바구니 담기" 버튼까지 볼 수 있는 화면을 만들었어요.

**비유** — 물건에 붙어 있는 설명서나 태그를 자세히 들여다보는 것과 같아요.

`상세 페이지` `리뷰 목록`

## 7. 카트(장바구니) 만들기

**한 일** — 담은 상품의 수량을 바꾸거나 지울 수 있고, 상품 금액·배송비·총 결제금액까지 자동으로 계산해주는 화면을 만들었어요. 브라우저에 저장해서 새로고침해도 사라지지 않게 했어요.

**비유** — 마트에서 실제 카트를 밀고 다니면서 물건을 담고 빼고 하는 것과 똑같아요.

`장바구니 상태` `localStorage`

## 8. 손님 눈높이 맞추기 (반응형 점검)

**한 일** — 휴대폰, 태블릿, 컴퓨터 화면 크기에서 다 확인해봤어요. 버튼이 손가락으로 누르기엔 너무 작았던 부분을 찾아서 크기를 키웠어요.

**비유** — 키가 다른 손님들이 와도 다 편하게 이용할 수 있도록 진열대 높이를 확인하는 것과 같아요.

`반응형 디자인` `터치 영역`

## 9. 작업 기록 저장하기

**한 일** — 지금까지 만든 것을 Git이라는 도구로 스냅샷 찍듯이 저장(커밋)하고, GitHub이라는 온라인 저장소에 올렸어요(푸시).

**비유** — 게임할 때 저장 포인트를 만들어두는 것과 같아요. 나중에 문제가 생겨도 저장해둔 지점으로 되돌아갈 수 있어요.

`git commit` `git push`

## 10. 진짜 가게 문 열기 (배포)

**한 일** — "Vercel"이라는 서비스에 올려서, 인터넷 주소만 알면 누구나 들어올 수 있는 진짜 사이트로 만들었어요.

**비유** — 지금까지는 우리끼리만 볼 수 있는 모델하우스였다면, 이제 진짜 손님을 받는 가게 문을 연 거예요.

`Vercel` `배포`
