# 개발 여정 문서

라오스 커피 쇼핑몰이 만들어진 10단계 과정을, 코드를 몰라도 이해할 수 있도록 정리한 문서입니다. 세 가지 형식으로 저장되어 있습니다.

- [`dev-journey.md`](./dev-journey.md) — 마크다운 버전 (GitHub에서 바로 읽기 좋음)
- [`dev-journey.html`](./dev-journey.html) — 꾸며진 웹페이지 버전 (다운로드해서 브라우저로 열기)
- [`dev-journey.docx`](./dev-journey.docx) — 워드 문서 버전

## 구글 Docs로 열기

`dev-journey.docx`는 구글 드라이브에 업로드해서 그대로 구글 Docs로 열 수 있어요.

1. [drive.google.com](https://drive.google.com)에 접속해 로그인
2. `dev-journey.docx` 파일을 드라이브에 업로드
3. 업로드된 파일을 더블클릭하거나, 마우스 우클릭 → "연결 앱" → "Google 문서" 선택

한 번 구글 Docs로 열어서 저장하면 이후에는 원본 워드 파일과 별개로 구글 문서 사본이 만들어집니다.
